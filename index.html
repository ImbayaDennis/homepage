A brief overview

This project serves as an amalgamation of various web development concepts that we have been learning throughout the entirety of our first year of our degree. We have employed the use of HTML, CSS, SCSS and Javascript together with some of the most popular libraries and frameworks such as bootstrap and jQuery for our front end designs and PHP and SQL for the backend and database development.
